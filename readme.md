# 🧩 Modos do Copiloto (Ask, Edit, Plan, Agent e Study)

![Prompt](https://img.shields.io/badge/Prompt-engineering-yellow)

Este repositório agrupa exemplos de prompts e fluxos para o Copilot/IA, com foco em 5 modos de uso. Você pode escolher o modo ideal para a atividade atual e usar o prompt completo ou a versão curta.

## Como usar
1. Abra o modo desejado no Copilot: **Ask**, **Base**, **Plan**, **Agent**, ou **Study**.
2. Copie o prompt de `prompts/prompt-<modo>.md` (ou versão curta em `prompts/prompt-<modo>-short.md`).
3. Cole na interface do assistente e envie.
4. Ajuste instruções específicas do seu contexto (nome de arquivo, objetivo, linguagem etc.).

---

## 🚀 Modos disponíveis

### 1) Ask
- Objetivo: obter explicações, diagnósticos e esclarecimentos.
- Não altera código.
- Indicado para dúvidas sobre erros, arquitetura, chamadas de API e conceitos.
- Arquivos de prompt:
  - `prompts/prompt-ask.md`
  - `prompts/prompt-ask-short.md`

---

### 2) Base
- Objetivo: modificar código existente com base em instruções específicas da minha própria personalidade.
- Use para refatorar, corrigir bugs, melhorar desempenho, atualizar estilo.
- Arquivos de prompt:
  - `prompts/prompt-base.md`
  - `prompts/prompt-base-short.md`

---

### 3) Plan
- Objetivo: planejar a implementação antes de codificar.
- Ideal para features complexas ou reestruturação.
- Arquivos de prompt:
  - `prompts/prompt-plan.md`
  - `prompts/prompt-plan-short.md`

---

### 4) Agent
- Objetivo: automação de tarefas em várias etapas, como um assistente autônomo.
- Pode gerar/editar múltiplos arquivos e manter contexto de execução.
- Arquivos de prompt:
  - `prompts/prompt-agent.md`
  - `prompts/prompt-agent-short.md`

---

### 5) Study
- Objetivo: aprendizado guiado e progressivo sobre um tema técnico.
- Excelente para resumo de conceitos, exemplos passo a passo e perguntas reflexivas.
- Arquivos de prompt:
  - `prompts/prompt-study.md`
  - `prompts/prompt-study-short.md`

---

## 📌 Observações do projeto
- Todos os prompts estão em `prompts/`.
- Use a versão curta para ciclos rápidos e a versão completa para workflows mais detalhados.
- Atualize os prompts de acordo com o contexto do seu repositório (framework, linguagem, arquitetura, camada de backend/frontend etc.).

---

## 🧾 Resumo rápido
- `Ask` = entender
- `Plan` = projetar
- `Base` = transformar
- `Agent` = executar/automatizar
- `Study` = aprender
